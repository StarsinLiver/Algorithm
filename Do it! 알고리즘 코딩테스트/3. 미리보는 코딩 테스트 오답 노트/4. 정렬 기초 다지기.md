## 정렬 기초 다지기

### 오름차순 정렬

오름차순 정렬은 데이터를 가장 작은 값부터 큰값으로 나열하는 과정을 의미합니다. 즉, 정렬된 결과는 가장 작은 값이 첫 번째 위치에, 그다음 작은 값이 두 번째 위치에 오는 방식입니다. c++에서는 다음과 같이 sort()함수를 사용하여 오름차순 정렬을 간단하게 구현할 수 있습니다.

```c
#include <algorithm>
#include <iostream>
#include <cstdlib>

using namespace std;

int main(int argc, char *argv[]) {

    int A[] = {5,3,2,4,1};
    int n = sizeof(A) / sizeof(A[0]);
    sort(A , A + n);
    for (int i = 0; i < n; i++) {
        cout << A[i] << " ";
    }

    return EXIT_SUCCESS;
}
```

### 실행결과

```
1 2 3 4 5
```

실행결과를 살펴보면, sort() 함수를 통해 A 배열의 값이 오름차순으로 정확히 정렬된 결과를 확인할 수 있습니다.

### 내림차순 정렬

내림차순 정렬은 데이터를 가장 큰 값부터 시작하여 점차 작은 값으로 나열하는 과정을 의미합니다. 즉, 오름차순 정렬과 반대로 가장 큰 값이 첫번째 위치에, 그다음 큰 값이 두번째 위치에 오는 방식입니다.
C++ 에서는 다음과 같이 sort()함수의 3번째 파라미터에 내림차순 비교 함수 객체인 greater<int>()를 추가하여 내림차순 정렬을 쉽게 구현할 수 있습니다.

```c
#include <algorithm>
#include <iostream>
#include <cstdlib>

using namespace std;

int main(int argc, char *argv[]) {

    int A[] = {5,3,2,4,1};
    int n = sizeof(A) / sizeof(A[0]);
    sort(A , A + n, greater<int>());
    for (int i = 0; i < n; i++) {
        cout << A[i] << " ";
    }

    return EXIT_SUCCESS;
}
```

### 실행결과

```
5 4 3 2 1
```
