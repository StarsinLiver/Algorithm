## 다중 조건 정렬 익히기

문제를 다루다 보면 데이터를 여러 기준에 따라 정렬해야 하는 상황이 주주 등장합니다. 예를 들어 성적을 정렬할 때 먼저 영어 점수를 기준으로 하되, 영어 점수가 같을 경우에는 수학 점수를 기준으로 하고 싶을 수 있습니다. 이럴 때 다중 조적ㄴ 정렬을 사용하면 여러 조건을 동시에 적용하여 원하는 순서대로 데이터를 정렬할 수 있습니ㅏㄷ.

C++ 에서는 sort()에 사용자 정의 비교 함수를 입력하여 다중 조건 정렬을 구현할 수 있습니다. 또한 구조체 내부에서 비교 연산자 오버로딩을 통해서도 가능합니다.

### sort()에 사용자 정의 비교 함수 입력하기

```c
#include <algorithm>
#include <iostream>
#include <cstdlib>

using namespace std;

struct Score {
    int english, math;
};

bool compare(const Score &a, const Score &b) {
    if (a.english != b.english) {
        return a.english > b.english;
    }
    return a.math > b.math;
}

int main(int argc, char *argv[]) {
    Score score[] = {
        {80, 100},
        {100, 50},
        {70, 100},
        {80, 90},
    };

    int n = sizeof(score) / sizeof(score[0]);
    sort(score, score + n, compare);

    for (int i = 0; i < n; ++i) {
        cout << "영어 : " << score[i].english << ", 수학 : " << score[i].math << "\n";
    }

    return EXIT_SUCCESS;
}
```

#### 실행결과

```
영어 : 100, 수학 : 50
영어 : 80, 수학 : 100
영어 : 80, 수학 : 90
영어 : 70, 수학 : 100
```

실행 결과를 보면, Score 배열이 영어 점수를 기준으로 내림차순 정렬되었습니다. 또한 영어 점수가 같은 경우네느 수학 점수를 기준으로 내림차순 정렬된 것을 확인할 수 있습니다.

### 구조체 내부에서 비교 연산자 오버로딩 하기

다음은 구조체 내부에서 비교 연산자 오버로딩을 이용하여 성적을 정렬한 예시입니다. 마찬가지로 2가지 조건을 기준으로 정렬하는데, 이번에는 수학 점수를 우선 기준으로 하고 수학 점수가 같을 경우에는 영어 점수로 정렬해 보겠습니다.

#### 구조체 내부에서 비교 연산자 오버로딩을 이용한 다중 조건 정렬

```c
#include <algorithm>
#include <iostream>
#include <cstdlib>

using namespace std;

struct Score {
    int english, math;
    bool operator <(const Score &other) const {
        if (math != other.math) {
            return math > other.math;
        }
        return english > other.english;
    }
};

int main(int argc, char *argv[]) {
    Score score[] = {
        {80, 100},
        {100, 50},
        {70, 100},
        {80, 90},
    };

    int n = sizeof(score) / sizeof(score[0]);
    sort(score, score + n);

    for (int i = 0; i < n; ++i) {
        cout << "영어 : " << score[i].english << ", 수학 : " << score[i].math << "\n";
    }

    return EXIT_SUCCESS;
}
```

#### 실행결과

```
영어 : 80, 수학 : 100
영어 : 70, 수학 : 100
영어 : 80, 수학 : 90
영어 : 100, 수학 : 50
```

이번에는 수학 점수를 기준으로 내림차순 정렬하고, 수학 점수가 같은 경우에는 영어 점수를 기준으로 내림차순 정렬된 결과를 확인할 수 있습니다.
